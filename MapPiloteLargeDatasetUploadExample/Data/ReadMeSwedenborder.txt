The default Geopackage for the Swedesh border is SwedenBorder.gpkg in this folder.
It a rough outline of the Swedish border, with only one feature.
Containing 400 vertices, it is a small file (28 kB) that is easy to work with in examples.
It is not suitable for production use, as it is not accurate enough.

If you need a more accurate border use the SwedenBorderHighResolution.gpkg
It contains 3034 vertices and is 212 kB.
